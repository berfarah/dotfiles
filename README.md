# Bernardo's Dotfiles

Get going quickly by copy-pasting the below into your terminal:

```sh
bash -c "$(curl -L https://raw.githubusercontent.com/berfarah/dotfiles/master/bin/setup.sh)"
```

### Caveats

- There are password prompts during the installation process for `sudo` and
  macOS settings configuration.
- The setup script will prompt you for a computer name.
